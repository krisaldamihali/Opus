# Opus

Opus is a React JS, Firebase, and Node.js-based social networking prototype designed to provide users with a safe, simple, and reliable virtual space for sharing posts, building profiles, and engaging in a structured online community.

## Tech Stack

- React JS
- React Router
- Firebase
- Node.js
- Bootstrap
- React Icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm start
```

The app will open at:

```text
http://localhost:3000
```

## Scripts

- `npm start` - starts the development server
- `npm test` - runs tests
- `npm run build` - creates a production build

## Project Structure

- `src/` - React components, pages, styles, and Firebase config
- `public/` - static files used by the app
